
사용자의 일일 컨디션과 운동 기록을 기반으로
오늘 수행하기에 적절한 운동 루틴 또는 회복 전략을 추천하는 AI 헬스케어 앱

앱의 목적
-> 헬스를 꾸준히 하는 사람 대부분이 겪는 문제
오늘 운동해도 되는 날인지, 피곤하지만 습관적으로 운동함, 어제 한 운동을 고려하지 않고 과부하가 걸림, 회복이 필요한 날에도 뭘 해야 할지 모름

1) 사용자 체크인 (Input)

사용자는 매일 간단한 정보를 입력한다:

오늘 컨디션 (좋음 / 보통 / 나쁨)

수면 시간

근육통 여부

어제 운동 부위

오늘의 목표 (근비대 / 유지 / 회복)

2) AI + 규칙 기반 판단 (Process)

앱은 두 단계로 판단한다:

규칙 기반 로직

수면 부족 or 근육통 → 회복 우선

동일 부위 연속 운동 → 다른 부위 추천

AI 추천

오늘 상태에 맞는 운동 루틴 구성

운동 종목, 세트/반복, 회복 팁 제공

-> AI는 무작위 추천이 아니라,
사전에 걸러진 조건 위에서만 작동한다.

3) 추천 결과 제공 (Output)

사용자는 다음 중 하나를 받는다:

 운동 추천 루틴

 저강도 / 볼륨 조절 루틴

 회복 중심 가이드 (스트레칭, 가벼운 유산소 등)

 이 앱의 차별점 (포트폴리오 포인트)
 흔한 헬스 앱
“AI가 추천해줘요”
기준 불명확
사용자 맥락 고려 부족

✅ 이 앱

사용자 상태 → 판단 근거가 명확
AI는 “보조 역할”
-> 헬스를 실제로 해본 사람이 설계한 로직
- 면접관이 좋아하는 구조:

“도메인 이해 + 기술 활용”

🧩 현재 범위 (개인 프로젝트 기준)

이 앱은 일부러 범위를 제한했다.
 칼로리 계산
 식단 관리
 체중 트래킹
👉 목적은 하나:
-> 오늘 운동할지 말지, 한다면 어떻게 할지
